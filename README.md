# Food Hub

A simple console-based Java project built in IntelliJ IDEA to manage food orders.  
This project allows users to generate bills, display them, and select a payment method.

---

## Features
- Generate Bill for ordered items
- Display Bill
- Choose Payment Mode (Cash, Card, or Digital)
- Simple and beginner-friendly console interface
